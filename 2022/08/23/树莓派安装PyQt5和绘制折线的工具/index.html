树莓派中安装PyQt5和绘制折线的工具

1.安装PyQt5

注意事项：1.不能通过pip3安装，有报错;2.要先换源之后再使用apt-get安装

步骤：

  1. 更换软件源：sudo nano /etc/apt/sources.list
  2. 更换这个软件源：deb http://mirrordirector.raspbian.org/raspbian/ stretch main contrib non-free rpi
  3. 更新：sudo apt-get update
  4. 安装pyqt5：sudo apt-get install python3-pyqt5
  5. 检验：from PyQt5.Qt import *

2.安装PyQt5.QtChart(在Qt5中绘制表格工具)

1.执行：sudo apt install python3-pyqt5.QtChart
2.检验：from PyQt5.QtChart import QChart
